# Larena Dataview

Developer vertical slice: package-owned source providers can now produce a safe, paginated Table projection used by the real Pages and Users admin screens. This is not a claim of full Dataview or production readiness.

Universal data presentation layer that can render the same structured dataset as table, kanban, calendar, gantt, cards, tree or other reusable views without reimplementing the same UI for every package.

This repository is a clean Larena pre-codegen baseline. Package implementation code has not started.

Canonical specifications are in `simai/larena-specs`.
